Modular WordPress toolkit plugin for client websites.

== Description ==

Dr.Slon Toolkit is a modular plugin for practical client-site tasks.

Planned modules:
- Transliteration
- Disable Comments
- Cleanup
- Hide Login
- Redirect Manager
- Login Attempts
- REST API Control
- IndexNow
- Sitemap
- Update Controls

== Installation ==

1. Upload the plugin to the /wp-content/plugins/ directory.
2. Activate the plugin through the WordPress admin panel.
3. Open Dr.Slon Toolkit in the admin menu.
